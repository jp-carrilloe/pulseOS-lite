## Purpose

## Architecture Overview
Acme Inc keeps client operating knowledge in structured Markdown documents, uses a local SQLite index for retrieval, and exposes a graph view to inspect document relationships. The sample stays in the repo for testing, but it is excluded from the curated index and graph data.
