## Purpose
This directory centralizes all efforts related to securing capital for [CLIENT_NAME]. It contains the strategic narrative, the tracking of investor relationships, and the documents required for due diligence.

## Directory Structure

| Folder/File | Purpose |
|-------------|---------|
| `Fundraising_Strategy.md` | Tier-one fundraising strategy, operating process, and negotiation guardrails. |
| `Fundraising_Agent.md` | In-folder operating reference for `@RAISE` (Fundraising Agent). |
| `402.1_Fundraising_Process_Tracker/` | Tracks the physical stages of the fundraising funnel. |
| `402.2_Data_Room/` | Repository for all documents shared with investors. |
| `402.3_Investor_CRM/` | Detailed profiles and interaction logs for potential investors. |

## How to Use This Folder
102. **Define Strategy**: Start with `Fundraising_Strategy.md` to lock round design, targets, and process timing.
201. **Run Agent Cadence**: Use `Fundraising_Agent.md` to execute weekly operating rhythm and stage exit rules.
202. **Build the Pipeline**: Populate `402.1_Fundraising_Process_Tracker` with target investors and stage progression.
203. **Prepare the Data Room**: Collect all historical and forward-looking documents in `402.2_Data_Room`.
301. **Manage Relationships**: Use `402.3_Investor_CRM` to track every touchpoint with specific investors.
