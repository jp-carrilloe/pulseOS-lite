# 402 Fundraising — Strategy & Assets

## Purpose
Centralizes Tintto's current investor narrative, founder formation strategy, teaser materials, and capital-readiness assets. At this stage, the folder supports **stealth founder formation and selective investor conversations**, not a broad formal fundraising process.

## Directory Structure

- `402_Fundraising_Agent.md` — governance spec for the 402 Fundraising Agent
- `402.0_Fundraising_Documents/` — master roadmap, investor discussion brief, teaser materials, and the later-stage full fundraising deck
- `402.0_Fundraising_Documents/06_LinkedIn_Posts/` — founder LinkedIn post drafts and public-facing narrative working files
- `402.1_Fundraising_Strategy/` — founder formation strategy, investor targeting (including `402.1.2_Strategic_Angels.md`), and application drafts
- `402.2_Data_Room/` — disclosure structure and investor FAQ
- `402.3_Investor_CRM/` — investor pipeline, scouting brief, and enrichment scripts

## How to Use This Folder

1. **Lock the founder formation narrative** in `402.1_Fundraising_Strategy/402.1_Fundraising_Strategy.md`
2. **Run the agent cadence** using `402_Fundraising_Agent.md`
3. **Use teaser-safe materials only** from `402.0_Fundraising_Documents/` for early conversations
4. **Use the full fundraising deck later** only when post-team / pre-formal-round disclosure is appropriate
5. **Build the right investor network** in `402.3_Investor_CRM/` with talent access and strategic value in mind
6. **Keep the later formal round warm** without signaling that a broad process is open

## Deck Production Rules

- Teaser materials are for selective founder formation, strategic angel, and high-conviction referral conversations.
- Full fundraising materials are for later-stage diligence only.
- Keep the teaser tone direct and technical. Do not add generic AI language, inflated adjectives, or consultant-style framing.
- End every teaser deck with a specific CTA. The default ask is one strong intro to talent, strategic angels, or operators deploying agents.
- Do not include empty screenshot placeholders. Use real product visuals, redacted visuals, or a clean product-flow diagram.
- Keep one idea per slide and verify every slide in browser before sharing.
- For HTML decks, keep a dated backup and changelog beside the deck file before material edits.
- Follow the canonical presentation rules in `102_Corporate_Strategy_and_Foundation/102.4_Brand_Identity/102.4.3_Presentation_Design_Standards.md`.

## Readiness Gate
No formal fundraising outreach launches without:
- [ ] Founding team substantially assembled
- [ ] Stealth teaser materials aligned and approved
- [ ] Full fundraising deck aligned for later-stage disclosure
- [ ] Investor pipeline segmented into talent connectors, anchor angels, and later-round targets
- [ ] Data room structure updated for post-team diligence
- [ ] Formal round thesis, instrument, and dilution approach defined when timing is right
