# Data Room Index — Tintto

## Purpose
Defines the disclosure structure for Tintto's current stealth investor process. The data room is intentionally staged: teaser materials are available now for selective conversations, while fuller diligence materials are reserved for the period after the founding team is assembled and the company is preparing for a formal pre-seed.

## Disclosure Timeline

- **Targeting / Intro Requested**
  - No materials by default
- **Selective First Conversation**
  - Investor discussion one-pager
  - Stealth one-pager
  - Teaser deck
  - Founder formation memo
- **Talent / Anchor Angel Discussions**
  - Investor FAQ
  - VC objection handling memo
  - Additional high-level context only as needed
- **Post-Team / Pre-Formal Round**
  - Full fundraising deck
  - Expanded product and roadmap materials
  - Team and hiring materials
  - Financial planning documents
- **Formal Diligence**
  - Full data room access
