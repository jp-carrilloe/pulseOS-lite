React components and design templates that can be bulk-generated programmatically.

## Files
- `facebook_ad_square.jsx` — React component for Facebook square ad format
- `landing_page_layout.jsx` — React template for SEO landing pages

## How It Works
Designs are defined in code (React/JSX). The system uses `html-to-canvas` to screenshot the rendered component into a downloadable PNG for Facebook Ads or other platforms.
