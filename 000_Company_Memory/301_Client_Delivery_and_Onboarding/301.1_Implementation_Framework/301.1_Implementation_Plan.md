## Operating Model
- [ ] Follow a gated 90-day sequence: Day 7 architecture approval -> Day 14 baseline readiness -> Day 21 launch -> Month 2 scaling -> Month 3 handover. (Owner: Delivery Lead)

- [ ] No activation before deliverability, tracking, and CRM sync checks pass. (Owner: Technical Lead)

- [ ] Implement UTM schema and naming standards (`301_Client_Delivery_and_Onboarding/`). (Owner: Analytics Lead, Target: [DAY_14_DATE])

## Launch Gate (Day 21)
- [ ] Go/No-Go checklist:
  - [ ] CRM sync validated (Owner: RevOps Engineer)
  - [ ] Deliverability checks passed (Owner: Infrastructure Lead)
  - [ ] KPI dashboard live (Owner: Analytics Lead)
  - [ ] Sequence QA approved (Owner: QA Lead)
  - [ ] Client sign-off recorded (Owner: Account Lead)
- [ ] Launch first outreach and supporting content motions. (Owner: Demand Gen Lead, Target: [DAY_21_DATE])
- [ ] Log first-week baseline performance snapshot. (Owner: Analytics Lead, Target: [DAY_21_PLUS_7_DATE])

## Month 2: Scaling and Optimization
- [ ] Increase channel volume only on segments/channels passing baseline thresholds. (Owner: Growth Lead)
- [ ] Run weekly optimization cycles using KPI + experiment evidence. (Owner: Analytics Lead)
- [ ] Update optimization log weekly (`302_Analytics_and_Performance_Intelligence/302.3_Optimization_Logs/`). (Owner: Analytics Lead)
- [ ] Hold monthly deep-dive review and document decisions. (Owner: Delivery Lead)
- [ ] Route approved strategic changes into GTM decision log. (Owner: Strategy Lead)

## Month 3: Stabilization and Handover
- [ ] Consolidate winning playbooks and deprecate underperforming variants. (Owner: Growth Lead)
- [ ] Deliver client operator training (response handling, CRM hygiene, workflow ownership). (Owner: Client Success Manager)
- [ ] Perform final systems audit (data flow, attribution, deliverability). (Owner: Technical Lead)
- [ ] Deliver handover pack:
  - [ ] Active workflows and SOPs (Owner: Operations Lead)
  - [ ] KPI baseline and latest trends (Owner: Analytics Lead)
  - [ ] Open risks and mitigation plan (Owner: Delivery Lead)
  - [ ] Next 90-day recommendations (Owner: Strategy Lead)
- [ ] Confirm transition model (advisory cadence, escalation path, owners). (Owner: Account Lead)

## Governance Cadence
- [ ] Weekly: async pulse (pipeline, conversion, blockers, experiments). (Owner: Delivery Lead)
- [ ] Monthly: live performance review with optimization decisions. (Owner: Delivery Lead)
- [ ] Quarterly: strategy recalibration and ICP refresh. (Owner: Strategy Lead)

## Risk and Escalation Rules
- [ ] Escalate delivery-to-promise misalignment to @ARK immediately. (Owner: Delivery Lead)
- [ ] Escalate technical blockers to Infrastructure owner with impact and workaround. (Owner: Technical Lead)
- [ ] Escalate scope changes requiring legal/commercial updates through SOW change process. (Owner: Account Lead)
